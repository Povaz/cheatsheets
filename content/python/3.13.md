## [card whats-new] What's new in 3.13 (vs 3.12)

| feature              | desc                                        | detail                                                                                      |
| -------------------- | ------------------------------------------- | ------------------------------------------------------------------------------------------- |
| free-threaded build  | experimental no-GIL build (PEP 703)         | Opt-in build; officially *supported* only from 3.14 (PEP 779).                              |
| JIT compiler         | experimental copy-and-patch JIT             | Opt-in at build time.                                                                       |
| new REPL             | multi-line editing, history, color          | Implemented in pure Python; syntax highlighting lands in 3.14.                              |
| color tracebacks     | on by default in interactive mode           | Respects `NO_COLOR` / `FORCE_COLOR`.                                                        |
| `dbm.sqlite3`        | new default `dbm` backend                   | Migration path off the older `gdbm` / `ndbm` backends.                                      |
| typing `TypeIs`      | narrowing user-defined type guards          | PEP 742.                                                                                    |
| `warnings.deprecated` | programmatic deprecations                  | PEP 702.                                                                                    |

> [tip] If you're on 3.13, jumping to 3.14 gives you deferred annotations, template strings, and an official no-GIL build.
